# PR Reviewer Service

Сервис для управления пользователями и пул-реквестами, включая автоматическое назначение ревьюеров.

## Структура проекта

cmd/                — Точка входа, запуск HTTP-сервера  
domain/             — Доменные модели и ошибки  
repository/         — Работа с PostgreSQL  
service/            — Бизнес-логика  
handler/            — HTTP-хендлеры  
migrations/         — SQL-миграции  
docker-compose.yml  — Конфигурация docker compose
Makefile            — Команды для запуска

## Запуск проекта

Поднять приложение, PostgreSQL и запустить миграции:
```bash
make up
```

После запуска сервис доступен на 8080 порту
